###### 삼성청년아카데미(SSAFY) 모듈형 특강

# React + TypeScript + Vite 구성

차세대 프론트엔드 도구 Vite를 사용해 React 프로젝트를 스캐폴딩하고 여러 도구를 체인 구성하는 방법을 학습합니다.

- React
- TypeScript
- Tailwind CSS
- Stylelint
- Prettier
- ESLint
- Vitest

## React 컴포넌트 디자인

- 상태를 가지지 않는(Stateless) 컴포넌트
- 상태를 가지는(Stateful) 컴포넌트
- 컴포넌트 테스트 (TDD)
